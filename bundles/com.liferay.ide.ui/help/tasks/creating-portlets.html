Task Requirements

Below are the requirements needed to complete this tutorial:

  1. At least one new Liferay Project that is a portlet plugin type. See the documentation for creating new Liferay projects here: Creating New Liferay Projects
  2. Liferay Portal 6.0 or greater with Tomcat bundle. See this documentation on getting started setting up Liferay Tomcat bundle: Getting Started

New Liferay Portlet Wizard

Use this procedure below to create a new Liferay Portlet wizard and add it to a Liferay project.

  1. From the workbench window select File > New Project... > New Liferay Project
