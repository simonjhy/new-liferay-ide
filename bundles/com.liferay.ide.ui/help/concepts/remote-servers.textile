h1. Remote Liferay Servers

p. Normally Liferay project development involves using a local Liferay server bundle that is launched from within Eclipse.  However, Liferay IDE also supports the use of remote Liferay servers through the *Remote Liferay Server* adapter.  See the "Using Remote Servers":../tasks/using-remote-servers.html
